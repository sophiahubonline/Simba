Télécharger les polices et placez-les ici

But: pour que le site n'aille pas chercher les polices sur Internet, placez les fichiers de police suivants dans ce dossier `content/fonts/` :

Fichiers attendus (noms utilisés par CSS):
- Montserrat-Regular.woff2
- Montserrat-Regular.ttf (optionnel fallback)
- Montserrat-600.woff2
- Montserrat-600.ttf (optionnel)
- Montserrat-700.woff2
- Montserrat-700.ttf (optionnel)
- Forum-Regular.woff2
- Forum-Regular.ttf (optionnel)

Où les obtenir:
- Montserrat: https://fonts.google.com/specimen/Montserrat — cliquez sur "Download family" pour obtenir les fichiers, puis extrayez et renommez les fichiers woff2/ttf selon les noms ci-dessus.
- Forum: https://fonts.google.com/specimen/Forum — idem, téléchargez la famille et placez le fichier Forum-Regular dans ce dossier.

Conseil: Gardez les fichiers woff2 (meilleure compression) et utilisez les ttf seulement comme fallback si nécessaire. Après avoir placé les fichiers, videz le cache du navigateur et rechargez la page.
